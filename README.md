# Basic Calculator

## Description

A simple calculator program written in Java that can perform basic arithmetic operations.

## Features

* Addition
* Subtraction
* Multiplication
* Division

## Usage

1. Compile the program:

   ```bash
   javac Calculator.java
   ```
2. Run the program:

   ```bash
   java Calculator
   ```

## Example

```
Enter first number: 10  
Enter second number: 5  
Choose operation (+, -, *, /): +  
Result: 15  
```
